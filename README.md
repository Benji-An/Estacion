# Estacion
Software profesional para análisis de precipitación y escorrentía superficial aplicando el Método Racional.

## Levantar la demo

1. Instala las dependencias con `pip install -r requirements.txt`.
2. Ejecuta `uvicorn main:app --reload`.
3. Abre `http://127.0.0.1:8000` y sube un archivo CSV para ver una vista previa.
